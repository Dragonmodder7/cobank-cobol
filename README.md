# Nubank feito em cobol

## Funcionalidades

- Criar contas bancárias
- Consultar saldo
- Depositar valores
- Realizar saques
- Listar contas existentes
- Limite de até 10 contas
 

## Pré-requisitos

Para compilar e rodar o projeto, é necessário ter o **GnuCOBOL** instalado.

### Linux (Debian/Ubuntu):
```bash
sudo apt install open-cobol
